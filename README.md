# Ray 承叡 | 個人網站專案

這是一個結合個人形象展示、技術技能說明與動態作品集的響應式網頁專案。

## 🚀 專案概述
本專案建立了一個專業的個人門戶網站，採用現代化的深色系（Navy & Aqua）視覺風格。核心特色在於結合了 **Google Apps Script (GAS)** 作為後端資料 API，讓影片作品與開發專案可以透過 Google Sheets 即時管理與更新，而無需頻繁修改前端程式碼。

## 🛠 技術棧
* **前端核心**：HTML5, CSS3, JavaScript (Vanilla JS)。
* **資料獲取**：非同步 Fetch API。
* **後端集成**：Google Apps Script (GAS)。
* **設計規範**：CSS Variables, Flexbox, CSS Grid 佈局。

## 📂 檔案結構
* `index.html`：定義網站的語意化結構，包含導覽列、英雄區、技能展示、影片/專案列表與聯絡資訊。
* `style.css`：負責全站的視覺美學，包含毛玻璃導覽列、自定義滾動條樣式與卡片互動動畫。
* `script.js`：核心邏輯處理，負責從 API 異步抓取資料、處理換行格式並動態渲染內容卡片。

## ✨ 核心功能說明

### 1. 動態資料渲染
網站會從 `VIDEO_API` 與 `PROJECT_API` 獲取資料：
* **自動過濾**：程式碼會檢查 `是否顯示` 欄位，僅渲染標記為顯示的內容。
* **狀態提示**：在資料載入期間或載入失敗時，會提供相應的 UI 提示（如「正在抓取...」或錯誤訊息）。

### 2. 精緻的 UI 效果
* **技能標籤系統**：針對 Python、HTML、CSS、JS、Node.js 與 SQL 等技術設計了專屬的邊框與懸浮發光色彩。
* **響應式佈局**：使用 `clamp()` 與媒體查詢（Media Queries），確保在手機與桌面裝置皆有流暢的閱讀體驗。
* **智慧卡片**：內容卡片包含標題、自動處理換行的說明文字，以及懸浮時會向上位移的動畫效果。

## 🔗 聯絡與社群連結
網站底部整合了多個平台連結，並設有專屬懸浮樣式：
* **Gmail**：直接開啟郵件撰寫介面。
* **GitHub**：查看原始碼與開源貢獻。
* **Facebook & YouTube**：社群動態與影片教學。
